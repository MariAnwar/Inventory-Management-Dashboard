# 📊 Inventory Management Dashboard (Excel)

## 🔷 Overview
This project presents a fully interactive Excel dashboard developed for a real client to monitor inventory performance, track missing items, and uncover operational inefficiencies.  

The dashboard enables store management to explore data dynamically, identify key loss patterns, and make informed, data-driven decisions.

---

## 🔷 Business Problem
The client faced challenges in:
- Tracking missing inventory across different days and shifts  
- Identifying which staff members or items contributed most to losses  
- Extracting clear insights from raw operational data  

---

## 🔷 Solution
An Excel-based dashboard was designed to:
- Centralize inventory tracking  
- Provide real-time filtering and analysis  
- Highlight key performance indicators (KPIs) and trends  

---

## 🔷 Key Features
- 📅 Track missing items by **day, weekday, and time of day**
- 👤 Analyze missing items by **responsible staff**
- 📦 Item-level analysis to identify **high-loss products**
- 🎯 KPI summary section for quick insights:
  - Total missing items  
  - Highest missing day  
  - Top missing item  
  - Staff with the highest missing records  
  - Top items and high-risk shifts  
- 🎛️ Interactive **slicers** for dynamic filtering:
  - Date  
  - Time of day  
  - Staff  
  - Item category  
- 📈 Visualizations:
  - Missing items trend over time  
  - Breakdown by shift and staff  
  - Top missing items  

---

## 🔷 Data Processing & Enhancements
To improve analysis accuracy, additional features were engineered:

- **Missing Items Column**  
  Calculates discrepancies between expected and actual quantities
  
- **Weekday Column**  
Extracts weekday names for trend analysis
